Congress Tweets Analysis

Exploring Political Discourse on Twitter from 2017 to 2023

What is the idea?

We analyze tweets from U.S. Congress members over a 6-year period to understand communication patterns, partisan framing, and topic trends. By treating individual Congress members as nodes and shared topics or interactions as edges, we can explore how discourse spreads and polarizes over time.

Goal for End User's Experience

  • Interactive Visualization: A network of Congress members linked by shared topics or hashtags.
  • Insights: Track how political language and framing evolve around key events like elections or crises.
  • Actionable Outputs: Identify polarization patterns, party alignment clusters, and communication strategy shifts.

Data Sources

  • CongressTweets (GitHub): Daily JSON files from June 2017 to July 2023

This dataset was selected because of its completeness and daily granularity. It spans 2,197 JSON files, totaling over 5 million tweets from verified Congressional accounts. Each tweet includes metadata such as time, text, user, and source.

View Dataset

Network & Text Integration

Congress members are represented as nodes. Edges are drawn based on co-occurrence of hashtags, topics, or sentiment similarity. NLP techniques like topic modeling and stance detection are used to extract themes and tone, allowing us to visualize discourse dynamics over time.
